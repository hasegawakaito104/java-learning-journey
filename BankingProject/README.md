## 🎯 プロジェクトの目的

1. **面接対策** - 銀行系システムの実務経験を証明
2. **技術習得** - Spring Boot + React の実装スキル
3. **案件獲得** - 月40万円のJava案件受注を目指す

## 📅 スケジュール

- **Week 1**: 基礎学習・設計
- **Week 2**: 実装・テスト
- **Week 3**: 面接準備・デプロイ

## 🚀 使用技術

- **Backend**: Java 17, Spring Boot 3.0
- **Frontend**: React 18, TypeScript
- **Database**: H2 Database（開発用）
- **Tools**: Git, VS Code, IntelliJ IDEA
